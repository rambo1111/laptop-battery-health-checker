~~~powershell
pip install beautifulsoup4 lxml pyinstaller
~~~

### Creating exe file

[`command_to_create_exe.md`](command_to_create_exe.md) First run this.
After running this command you'll see
```
Your directory/
|
├── build                 # Delete this
├── builder.spec                # Delete this
└── dist/
    └── builder.exe          # Your exe file
```
